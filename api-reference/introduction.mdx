## Welcome to Docta Capital API

The Docta Capital API provides access to market data and financial services. This API uses OAuth 2.0 client credentials flow for authentication.

<Card
  title="Base URL"
  icon="globe"
>
  https://api.doctacapital.com.ar
</Card>

## Getting Started

1. **Obtain Credentials**: Contact Docta Capital to get your `client_id` and `client_secret`
2. **Get Access Token**: Use the authentication endpoint to obtain an access token
3. **Make API Calls**: Include the access token in the Authorization header for all subsequent requests

## Authentication

The API uses OAuth 2.0 client credentials flow. You need to obtain an access token before making any API calls.

<Steps>
  <Step title="Request Access Token">
    Send a POST request to `/api/v1/auth/token` with your client credentials
  </Step>
  <Step title="Use the Token">
    Include the access token in the Authorization header: `Bearer {access_token}`
  </Step>
  <Step title="Token Expiration">
    Tokens expire after a certain period. You'll need to request a new token when the current one expires.
  </Step>
</Steps>

## Rate Limits

API requests are subject to rate limiting. Please refer to the response headers for current rate limit information.
