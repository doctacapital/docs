## Bienvenido a la API de Bonos de Docta

La API de Bonos de Docta brinda datos especializados de renta fija argentina: catálogo de instrumentos, detalle enriquecido, flujos de fondos, yields intradiarios y curvas históricas. La autenticación sigue el flujo de credenciales de cliente OAuth 2.0.

<Card
  title="URL Base"
  icon="globe"
>
  https://api.doctacapital.com.ar
</Card>

## Generar credenciales

1. **Generar Credenciales**: Generá tus credenciales en [doctacapital.com.ar](https://www.doctacapital.com.ar) (`client_id` y `client_secret`).
2. **Obtener Token de Acceso**: Usá tu `client_id` y `client_secret` para obtener un token de acceso. [Ver endpoint](/api-reference/endpoint/auth-token).

## Autenticación

La API utiliza el flujo de credenciales de cliente OAuth 2.0. Necesitas obtener un token de acceso antes de consultar instrumentos, cashflows o curvas.

<Steps>
  <Step title="Solicitar Token de Acceso">
    Envía una solicitud POST a `/api/v1/auth/token` con tus credenciales de cliente.
  </Step>
  <Step title="Usar el Token">
    Incluye el token de acceso en el header Authorization: `Bearer {access_token}`.
  </Step>
  <Step title="Expiración del Token">
    Los tokens expiran. Solicita uno nuevo al vencer para mantener el acceso a los datos de bonos.
  </Step>
</Steps>
