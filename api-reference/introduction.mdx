## Bienvenido a la API de Docta Capital

La API de Docta Capital proporciona acceso a datos de mercado y servicios financieros. Esta API utiliza el flujo de credenciales de cliente OAuth 2.0 para la autenticación.

<Card
  title="URL Base"
  icon="globe"
>
  https://api.doctacapital.com.ar
</Card>

## Comenzando

1. **Obtener Credenciales**: Contacta a Docta Capital para obtener tu `client_id` y `client_secret`
2. **Obtener Token de Acceso**: Usa el endpoint de autenticación para obtener un token de acceso
3. **Realizar Llamadas a la API**: Incluye el token de acceso en el header Authorization para todas las solicitudes posteriores

## Autenticación

La API utiliza el flujo de credenciales de cliente OAuth 2.0. Necesitas obtener un token de acceso antes de realizar cualquier llamada a la API.

<Steps>
  <Step title="Solicitar Token de Acceso">
    Envía una solicitud POST a `/api/v1/auth/token` con tus credenciales de cliente
  </Step>
  <Step title="Usar el Token">
    Incluye el token de acceso en el header Authorization: `Bearer {access_token}`
  </Step>
  <Step title="Expiración del Token">
    Los tokens expiran después de un cierto período. Necesitarás solicitar un nuevo token cuando el actual expire.
  </Step>
</Steps>

## Límites de Velocidad

Las solicitudes a la API están sujetas a límites de velocidad. Por favor, consulta los headers de respuesta para obtener información actual sobre los límites de velocidad.
