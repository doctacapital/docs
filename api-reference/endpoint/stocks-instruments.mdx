<Note>
  Lista instrumentos de acciones con paginación.
</Note>

## Parámetros de consulta

<ParamField query="limit" type="integer">
  Límite por página.
</ParamField>

<ParamField query="cursor" type="string">
  Cursor de paginación.
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/stocks/instruments?limit=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
