---
title: 'Intradiario'
openapi: 'GET /api/v1/cedears/market-data/{symbol}/intraday'
---

<Note>
  Devuelve datos intradiarios para un CEDEAR.
</Note>

## Parámetros

<ParamField path="symbol" type="string" required>
  Símbolo del CEDEAR (ej., "AAPL").
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/cedears/market-data/AAPL/intraday/" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
