---
title: 'Curvas de Rendimiento'
openapi: 'GET /api/v1/market_data/historicals/yield-curve'
---

<Note>
  Este endpoint obtiene las curvas de rendimiento (yield-curve) históricas o intradiarias de bonos para una fecha y sub-asset class determinado.
</Note>

## Parámetros de Consulta

<ParamField query="date" type="string" required>
  Fecha histórica específica en formato YYYY-MM-DD.
</ParamField>

<ParamField query="subasset_class" type="string" required>
  La clasificación de submercado para los datos de curva de rendimiento. Debe ser uno de los valores predefinidos.
</ParamField>

<ParamField query="specie" type="string">
  Especie de moneda. Usar "D" para Dólar (opcional).
</ParamField>

## Opciones de Clase de Subactivo

El parámetro `subasset_class` acepta los siguientes valores:

- `FIXED_RATE` - Soberano - Tasa Fija
- `CER` - Soberano - CER
- `DOLLAR_LINKED` - Soberano - Dólar Linked
- `TAMAR` - Soberano - Tamar
- `BADLAR` - Soberano - Badlar
- `HARD_DOLLAR` - Soberano - Globales / Bonares
- `BOPREAL` - Soberano - Bopreal / BCRA
- `ON_FIXED_RATE` - ON - Tasa Fija
- `ON` - ON - Hard Dólar
- `ON_CER` - ON - CER
- `ON_BADLAR` - ON - Badlar
- `ON_TAMAR` - ON - Tamar
- `ON_DOLLAR_LINKED` - ON - Dólar Linked
- `SUB_SOBERANO_FIXED_RATE` - Subsoberano - Tasa Fija
- `SUB_SOBERANO` - Subsoberano - Hard Dólar
- `SUB_SOBERANO_CER` - Subsoberano - CER
- `SUB_SOBERANO_BADLAR` - Subsoberano - Badlar
- `SUB_SOBERANO_TAMAR` - Subsoberano - Tamar

## Ejemplo de Solicitud

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/market_data/historicals/yield-curve?date=2025-01-02&subasset_class=HARD_DOLLAR&specie=D" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

## Respuesta Exitosa

<ResponseField name="data" type="array">
  Array de instrumentos financieros con sus datos de curva de rendimiento.
</ResponseField>

<ResponseField name="data[].ticker" type="string">
  El símbolo ticker del instrumento financiero.
</ResponseField>

<ResponseField name="data[].specie" type="string">
  La especie de moneda del instrumento.
</ResponseField>

<ResponseField name="data[].subasset_class" type="string">
  La clasificación de submercado del instrumento.
</ResponseField>

<ResponseField name="data[].maturity_date" type="string">
  La fecha de vencimiento del instrumento (null si no aplica).
</ResponseField>

<ResponseField name="data[].tir" type="number">
  La Tasa Interna de Retorno (TIR) del instrumento.
</ResponseField>

<ResponseField name="data[].tna_30_360" type="number">
  La Tasa Nominal Anual calculada con base 30/360 (null si no aplica).
</ResponseField>

<ResponseField name="data[].duration" type="number">
  La duración del instrumento en años.
</ResponseField>

<ResponseField name="data[].date" type="string">
  La fecha de los datos en formato YYYY-MM-DD.
</ResponseField>

<ResponseField name="date" type="string">
  La fecha histórica para la cual se obtuvieron los datos de curva de rendimiento.
</ResponseField>

<ResponseField name="filters" type="object">
  Los filtros aplicados a la consulta.
</ResponseField>

<ResponseField name="filters.specie" type="string">
  La especie de moneda utilizada en el filtro.
</ResponseField>

<ResponseField name="filters.subasset_class" type="string">
  La clasificación de submercado utilizada en el filtro.
</ResponseField>

## Notas

- Los datos están disponibles solo para días hábiles (excluyendo fines de semana y feriados)
- Las TIR (Tasas Internas de Retorno) se expresan como decimales (ej., 0.104657 = 10.4657%)
- La disponibilidad de datos históricos puede variar según la clase de subactivo y el rango de fechas
- Los datos incluyen instrumentos específicos con sus respectivos tickers y características financieras
