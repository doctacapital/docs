---
title: 'Curvas de rendimiento intradiarias'
openapi: 'GET /api/v1/bonds/yields/curve'
---

<Note>
  Obtiene la curva de rendimiento actual para una subclase de bonos.
</Note>

## Parámetros de consulta

<ParamField query="sub_asset_class" type="string" required>
  Subclase de activo (ej., "HARD_DOLLAR").
</ParamField>

<Info>
Valores válidos para <code>sub_asset_class</code>:

- <code>DOLLAR_LINKED</code>
- <code>HARD_DOLLAR</code>
- <code>CER</code>
- <code>BOPREAL</code>
- <code>FIXED_RATE</code>
- <code>SUB_SOBERANO</code>
- <code>ON</code>
- <code>TAMAR</code>
- <code>BADLAR</code>
- <code>ON_FIXED_RATE</code>
- <code>ON_CER</code>
- <code>ON_BADLAR</code>
- <code>ON_TAMAR</code>
- <code>ON_DOLLAR_LINKED</code>
- <code>SUB_SOBERANO_FIXED_RATE</code>
- <code>SUB_SOBERANO_CER</code>
- <code>SUB_SOBERANO_BADLAR</code>
- <code>SUB_SOBERANO_TAMAR</code>
</Info>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/bonds/yields/curve/?sub_asset_class=HARD_DOLLAR" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
