---
title: 'OHLC histórico'
openapi: 'GET /api/v1/cedears/market-data/{symbol}/ohlc/'
---

<Note>
  Devuelve datos OHLC históricos para un CEDEAR en un rango de fechas.
</Note>

## Parámetros

<ParamField path="symbol" type="string" required>
  Símbolo del CEDEAR (ej., "AAPL").
  
  <Info>
  Usa tickers normalizados.
  </Info>
</ParamField>

<ParamField query="from_date" type="string" required>
  Fecha de inicio (YYYY-MM-DD).
</ParamField>

<ParamField query="to_date" type="string" required>
  Fecha de fin (YYYY-MM-DD).
</ParamField>

<ParamField query="adjusted" type="boolean" required>
  Si se devuelven precios ajustados.
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/cedears/market-data/AAPL/ohlc/?from_date=2025-01-01&to_date=2025-10-09&adjusted=false" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
