---
title: 'Acciones - Buscar instrumentos'
openapi: 'GET /api/v1/stocks/instruments/search'
---

<Note>
  Busca instrumentos de acciones por texto.
</Note>

## Parámetros de consulta

<ParamField query="query" type="string" required>
  Texto de búsqueda.
</ParamField>

<ParamField query="limit" type="integer">
  Límite de resultados (predeterminado 10).
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/stocks/instruments/search?query=GGAL&limit=10" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
