---
title: 'CEDEARs - Detalle de instrumento'
openapi: 'GET /api/v1/cedears/instruments/{symbol}'
---

<Note>
  Devuelve los datos del instrumento CEDEAR especificado.
</Note>

## Parámetros de ruta

<ParamField path="symbol" type="string" required>
  Símbolo del CEDEAR (ej., "AAPL").
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/cedears/instruments/AAPL" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
