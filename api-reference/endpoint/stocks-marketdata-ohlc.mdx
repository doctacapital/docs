---
title: 'OHLC histórico'
openapi: 'GET /api/v1/stocks/market-data/{symbol}/ohlc'
---

<Note>
  Devuelve datos OHLC históricos para una acción.
</Note>

## Parámetros

<ParamField path="symbol" type="string" required>
  Símbolo de la acción (ej., "GGAL").
</ParamField>

<ParamField query="from_date" type="string" required>
  Fecha de inicio (YYYY-MM-DD).
</ParamField>

<ParamField query="to_date" type="string" required>
  Fecha de fin (YYYY-MM-DD).
</ParamField>

<ParamField query="adjusted" type="boolean">
  Si se devuelven precios ajustados.
</ParamField>

## Ejemplo

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/stocks/market-data/GGAL/ohlc/?from_date=2025-01-01&to_date=2025-10-09&adjusted=false" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
