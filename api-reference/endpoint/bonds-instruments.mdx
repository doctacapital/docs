---
title: 'Instrumentos'
openapi: 'GET /api/v1/bonds/instruments'
---

<Note>
  Lista instrumentos de bonos con soporte de paginación y filtrado opcional por subclase de activo.
</Note>

## Parámetros de consulta

<ParamField query="sub_asset_class" type="string">
  Filtra por subclase de activo (ej., "HARD_DOLLAR").
</ParamField>

<Info>
Valores válidos para <code>sub_asset_class</code>:

- <code>DOLLAR_LINKED</code>
- <code>HARD_DOLLAR</code>
- <code>CER</code>
- <code>BOPREAL</code>
- <code>FIXED_RATE</code>
- <code>SUB_SOBERANO</code>
- <code>ON</code>
- <code>TAMAR</code>
- <code>BADLAR</code>
- <code>ON_FIXED_RATE</code>
- <code>ON_CER</code>
- <code>ON_BADLAR</code>
- <code>ON_TAMAR</code>
- <code>ON_DOLLAR_LINKED</code>
- <code>SUB_SOBERANO_FIXED_RATE</code>
- <code>SUB_SOBERANO_CER</code>
- <code>SUB_SOBERANO_BADLAR</code>
- <code>SUB_SOBERANO_TAMAR</code>
</Info>

<ParamField query="limit" type="integer">
  Límite de elementos por página (predeterminado 50).
</ParamField>

<ParamField query="cursor" type="string">
  Cursor de paginación para obtener la siguiente página.
</ParamField>

## Ejemplo de Solicitud

<RequestExample>
```bash cURL
curl -X GET "https://api.doctacapital.com.ar/api/v1/bonds/instruments?limit=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</RequestExample>

## Respuesta Exitosa

<ResponseField name="data" type="array">
  Lista de instrumentos de bonos.
</ResponseField>

<ResponseField name="data[].ticker" type="string">
  Ticker del bono (ej., "S30N6").
</ResponseField>

<ResponseField name="data[].asset_class" type="string">
  Clase de activo del bono (ej., "BOND").
</ResponseField>

<ResponseField name="data[].sub_asset_class" type="string">
  Subclase de activo del bono (ej., "FIXED_RATE").
</ResponseField>

<ResponseField name="pagination" type="object">
  Información de paginación.
</ResponseField>

<ResponseField name="pagination.limit" type="integer">
  Límite de elementos por página.
</ResponseField>

<ResponseField name="pagination.has_next" type="boolean">
  Indica si hay más páginas disponibles.
</ResponseField>

<ResponseField name="pagination.next_cursor" type="string | null">
  Cursor para obtener la siguiente página. Es null si no hay más páginas.
</ResponseField>

<ResponseField name="pagination.total_items" type="integer">
  Número total de elementos disponibles.
</ResponseField>

<ResponseField name="filters" type="object">
  Filtros aplicados en la solicitud.
</ResponseField>

<ResponseField name="filters.sub_asset_class" type="string">
  Subclase de activo utilizada como filtro.
</ResponseField>

<ResponseField name="metadata" type="object">
  Metadatos de la respuesta.
</ResponseField>

<ResponseField name="metadata.total_records" type="integer">
  Número total de registros disponibles.
</ResponseField>
