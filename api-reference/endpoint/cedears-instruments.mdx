---
title: 'Instrumentos'
openapi: 'GET /api/v1/cedears/instruments/'
---

<Note>
  Lista los instrumentos CEDEAR disponibles con paginación.
  
  <Tip>
  Usa `limit` y `cursor` para paginar resultados.
  </Tip>
</Note>

## Parámetros de consulta

<ParamField query="limit" type="integer">
  Límite de elementos por página.
  
  <Info>
  Predeterminado 50.
  </Info>
</ParamField>

<ParamField query="cursor" type="string">
  Cursor de paginación.
  
  <Info>
  Usa el `next_cursor` de la respuesta anterior.
  </Info>
</ParamField>

## Ejemplo de solicitud

```bash
curl -X GET "https://api.doctacapital.com.ar/api/v1/cedears/instruments/?limit=50" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
